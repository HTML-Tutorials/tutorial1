Hallo liebe Zuschauer,

In diesem Video werde ich euch ein Tutorial machen wie ihr ganz einfach zu eurer Website mit HTML und CSS kommt ihr benötigt dafür nicht viel eigentlich nur zwei Sachen:

Browser
Editor (ich verwende ATOM welchen ihr unter Atom.io downloaden könnt)

Ich werde kostenlose Bilder von der Seite Unsplash.com verwenden ihr könnt natürlich auch eure eigenen Nehmen. Unsplash hat den Vorteil das man nur den Link kopieren muss und damit die Bilder nicht den Speicherplatz von dir verwenden.

Ich habe euch den Code auf GitHub hochgeladen dort könnt ihr ihn gerne nutzen bzw. kopieren.!

Wenn ihr eure Website später für andere zugänglich machen wollt müsst ihr euch bei einem Webspace anmelden wie z.B. bplaced.net für den Lokalen Gebrauch benötigt ihr dies aber nicht. Da reicht auch schon euer Browser aus.
